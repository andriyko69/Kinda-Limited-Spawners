# Kinda Limited Spawners

Kinda Limited Spawners changes mob spawners so they can only spawn a configurable total number of mobs.

Once a spawner reaches its limit, it stops working until it is upgraded with a nether star. An upgraded spawner becomes unlimited and uses a different texture to show its new state.

The goal is to make spawners feel more balanced in survival while still allowing players to invest resources into restoring full functionality.

## Configuration

This mod includes a config option for the maximum number of mobs a spawner can produce before it stops.

This allows modpack authors and server owners to decide how restrictive or generous spawners should be.

## Notes

This mod is designed for Minecraft 1.21.1 on NeoForge.

The unlimited state is stored directly on the spawner, so upgraded spawners keep their status.
